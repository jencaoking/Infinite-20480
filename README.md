# 小米手环10无限2048

一款专为小米手环10设计的经典2048益智游戏，采用**无限模式**，让玩家可以无限挑战，追求更高分数。

## 功能特性

- 🎮 **无限模式** - 永不结束，尽情享受游戏乐趣
- 🌓 **深色/浅色主题** - 一键切换，适应不同环境
- ✨ **流畅动画** - 方块移动动画效果
- ↩️ **撤销功能** - 后悔药，撤销上一步操作
- 💾 **本地存储** - 分数和进度自动保存

## 构建方法

参考官方文档：<https://iot.mi.com/vela/quickapp>

# **游戏玩法**

1. 通过上下左右滑动手势控制所有方块移动
2. 相同数字的方块碰撞时会合并成它们的和
3. 目标是不断合并，挑战更高的数字和分数
4. 棋盘填满时仍可继续操作，不会结束游戏

## 技术栈

- 框架：Vela OS (小米手环操作系统)
- 开发工具：aiot-toolkit
